# Desafio 06 - Anagramas

Programa em C que encontra anagramas utilizando um arquivo de palavras válidas (`words.txt`).

## Compilação

### Linux

```bash
gcc anagrama.c -o anagrama
```

### Windows (MinGW)

```bash
gcc anagrama.c -o anagrama.exe
```

## Execução

### Linux

```bash
./anagrama
```

### Windows

```bash
anagrama.exe
```

## Observação

O arquivo `words.txt` deve estar na mesma pasta do programa para funcionamento correto.
