# Desafio 07 - TAC

Programa em C que imprime as linhas de um arquivo em ordem inversa, semelhante ao comando `tac` do Linux.

## Compilação

### Linux

```bash
gcc tac.c -o tac
```

### Windows (MinGW)

```bash
gcc tac.c -o tac.exe
```

## Execução

### Linux

```bash
./tac arquivo.txt
```

### Windows

```bash
tac.exe arquivo.txt
```
