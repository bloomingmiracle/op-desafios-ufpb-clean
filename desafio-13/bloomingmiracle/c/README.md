# Desafio 13 - Passeio do Cavalo

Programa em C que encontra um caminho para o passeio do cavalo em um tabuleiro de xadrez.

## Compilação

### Linux

```bash
gcc cavalo.c -o cavalo
```

### Windows (MinGW)

```bash
gcc cavalo.c -o cavalo.exe
```

## Execução

### Linux

```bash
./cavalo
```

### Windows

```bash
cavalo.exe
```
