# Desafio 14 - Expressões Numéricas

Programa em C que avalia expressões matemáticas simples.

## Compilação

### Linux

```bash
gcc expressoes.c -o expressoes
```

### Windows (MinGW)

```bash
gcc expressoes.c -o expressoes.exe
```

## Execução

### Linux

```bash
./expressoes
```

### Windows

```bash
expressoes.exe
```
