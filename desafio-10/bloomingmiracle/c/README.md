# Desafio 10 - Turing Machine

Programa em C que simula uma Máquina de Turing simples utilizando arquivos de regras e dados.

## Compilação

### Linux

```bash
gcc turing.c -o turing
```

### Windows (MinGW)

```bash
gcc turing.c -o turing.exe
```

## Execução

### Linux

```bash
./turing
```

### Windows

```bash
turing.exe
```
