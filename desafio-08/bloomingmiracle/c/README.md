# Desafio 08 - Frações Simples

Programa em C que simplifica frações e imprime números inteiros ou frações reduzidas.

## Compilação

### Linux

```bash
gcc fracoes.c -o fracoes
```

### Windows (MinGW)

```bash
gcc fracoes.c -o fracoes.exe
```

## Execução

### Linux

```bash
./fracoes
```

### Windows

```bash
fracoes.exe
```
