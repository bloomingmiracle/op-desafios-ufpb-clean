# Desafio 04 - Contabilizar Peças de Xadrez

Programa em C que contabiliza a quantidade de peças em um tabuleiro de xadrez.

## Compilação

### Linux

```bash
gcc xadrez.c -o xadrez
```

### Windows (MinGW)

```bash
gcc xadrez.c -o xadrez.exe
```

## Execução

### Linux

```bash
./xadrez
```

### Windows

```bash
xadrez.exe
```
