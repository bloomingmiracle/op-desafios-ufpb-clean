# Desafio 11 - Primos em Pi

Programa em C que encontra sequências de números primos nas casas decimais de π.

## Compilação

### Linux

```bash
gcc primospi.c -o primospi
```

### Windows (MinGW)

```bash
gcc primospi.c -o primospi.exe
```

## Execução

### Linux

```bash
./primospi
```

### Windows

```bash
primospi.exe
```
