# Desafio 12 - Potências de 2

Programa em C que verifica se números são potências de 2.

## Compilação

### Linux

```bash
gcc potencias2.c -o potencias2
```

### Windows (MinGW)

```bash
gcc potencias2.c -o potencias2.exe
```

## Execução

### Linux

```bash
./potencias2
```

### Windows

```bash
potencias2.exe
```
